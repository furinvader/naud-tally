# Design Agent Index

Read this file before opening other files under [`docs/design/`](./).

## Purpose

- This directory holds the Penpot workflow guide, reusable design foundations, screen-brief templates, and committed SVG artifacts.
- Penpot remains the default design workspace, but the default repo handoff is the markdown brief plus a committed SVG in this directory tree.

## Load Rules

- Start with [`README.md`](README.md) for the current layout.
- Read [`foundations/agent-index.md`](foundations/agent-index.md) before opening files under [`foundations/`](foundations/).
- Open [`screen-brief-template.md`](screen-brief-template.md) only when you need to create a new task-specific brief.
- Read [`penpot-codex-workflow.md`](penpot-codex-workflow.md) for the operational workflow.
- Treat committed SVG files as the only repo-tracked design artifact format.
- Treat Penpot share links as optional review context, not required repo metadata.
