# T-006: Add Organizer Controls

## Status

- next

## Goal

- Let an organizer reset and export the tally safely without breaking the guest flow.

## Done when

- an organizer can reset the current tally with an explicit guard against accidental use
- an organizer can export the current tally in a simple portable format
- the guest-facing flow stays clear on a shared tablet
- tests cover the new reset and export behavior

## In scope

- organizer reset flow
- organizer export flow
- safeguards around destructive actions

## Out of scope

- authentication or permissions
- remote sync or upload destinations
- guest-based tracking
