# T-004: Scaffold the App

## Status

- done

## Goal

- Create the initial Angular app scaffold using the chosen stack.

## Done when

- an Angular app exists in the repo
- Angular Material is installed and configured
- NgRx SignalStore is available for the first feature state
- `npm start` runs the app locally from [`frontend/`](../../frontend/)
- `npm run build` produces a production build from [`frontend/`](../../frontend/)
- `npm test` runs the default unit-test setup from [`frontend/`](../../frontend/)

## In scope

- app scaffolding
- Angular Material setup
- initial [`frontend/`](../../frontend/) project structure
- pinned Node workflow

## Out of scope

- replacing the placeholder UI with the real product flow
- organizer functionality

- [`frontend/.nvmrc`](../../frontend/.nvmrc)

- Angular 21 standalone app scaffolded in the repo

- [`frontend/.nvmrc`](../../frontend/.nvmrc) works with `nvm use`
