# T-013: Apply the Design Foundations to the Angular Theme

## Status

- done

## Goal

- Apply the shared design foundation to the Angular theme and use the current tally feature as the proof target.

## Done when

- the global theme matches the shared foundation palette, typography, and shape direction
- repo-authored styles use `--nt-*` as the only app-owned CSS variable contract
- the tally feature consumes the shared theme with minimal local styling
- build and tests pass

## In scope

- global theme tokens
- self-hosted font loading
- Angular Material theme and component overrides
- tally feature theming
- frontend convention docs

## Out of scope

- dark mode
- a design-token export pipeline
- redesigning the upcoming guest-tab behavior

## Outcome

- created a shared frontend theme layer built from the design foundation
- introduced app-owned `--nt-*` theme tokens and reserved `--nt-<component>-*` for component-local variables
- wired `Manrope` and `Inter Tight` into the frontend through self-hosted public assets
- updated the tally feature to consume shared theme tokens instead of Angular Material system variables
- updated product and frontend docs so the new theming contract is explicit for future work
