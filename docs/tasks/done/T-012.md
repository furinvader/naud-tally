# T-012: Create a Reusable Design Foundation Board

## Status

- done

## Goal

- Create a reusable Penpot foundation screen that records the visual system for future `Naud Tally` design work.

## Done when

- a reusable foundation board exists in Penpot
- the board records the core color roles, typography roles, icon direction, and component direction
- the foundation reflects a coherent tablet-first product direction rather than an ad hoc palette sheet
- the visual direction is derived from the actual tally screen and future app usage rather than from a marketing-style reference layout
- the repo contains a matching markdown brief and SVG artifact for the foundation board
- future design tasks have a stable repo path they can use as baseline context

## In scope

- creating the Penpot foundation page and board
- defining the initial design tokens and role guidance
- recording the foundation in [`docs/design/`](../../design/)
- updating repo routing and decisions so later tasks treat the foundation as the baseline

## Out of scope

- fully designing every future product screen
- building a coded design-token pipeline
- introducing PNG artifacts
