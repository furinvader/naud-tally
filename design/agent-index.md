# Design Agent Index

Read this file before opening other files under [`design/`](./).

## Purpose

- This directory holds repo-tracked design artifacts that are easy for browsers, humans, and agents to inspect.
- Penpot remains the default design workspace, but committed artifacts here are the default repo handoff.

## Load Rules

- Start with [`README.md`](README.md) for the current artifact overview.
- Read the matching brief in [`../docs/penpot-example-drink-tally.md`](../docs/penpot-example-drink-tally.md) for goals, states, and implementation notes.
- Treat SVG exports as the preferred machine-readable snapshot when they are available.
- Treat PNG exports as quick visual previews for browser and repo review.
- Treat Penpot share links as optional review context, not required repo metadata.
